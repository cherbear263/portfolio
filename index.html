Flask Coffeeshop

Custom coffee ordering app for St Luke's Anglican School Year 10 catering program. This is currently being used for teachers to order coffees and desserts to be delivered to the right place. After interviewing many of the people involved in the hospitality program and brainstorming, I started coming up with a solution, starting with data flow. The goal was to save the teachers time by keeping track of payment as well as creating an easy-to-use ordering system that would automatically print labels and help manage inventory.The solution was done entirely from scratch, using Flask.

Some custom features are printable labels for takeaway containers and email reminders to teachers about payment. I added some fun stats on the admin home page and animated a graph of best-selling items with chart.js

The real app is hosted internally at the school, but I created a dummy site on pythonanywhere so that I could share it with you.

Real Social website made with Wagtail

This business website and blog was built with Wagtail (Django behind the scenes) for a custom CMS.

The design was modified from an html template I purchased. I broke it into reuseable templates and made it easy for my client to create his own blogs, pages and forms with StreamField (options for headings, images, video, subheadings, etc.).

I found that my biggest hurdle was deploying the site. Previously I used managed wordpress, which was very easy. For this site I had to learn a about gunicorn, nginx and adding an ssl certificate.
